#Christmas Congratulations!
This project is one for Christmas Day using laravel and bootstrap.

##Environment
		- mysql server: https://www.mysql.com/
====================================================
##How to Run?
		- clone project from git repository
			<command>: git clone https://github.com/kideveloper612/Christmas-laravel-bootstrap.git

		- load vendor directory
			<command>: composer install

		- connect to database
			Please make .env file in root directory and put database infomation.
				ex: DB_CONNECTION=mysql
					DB_HOST=127.0.0.1
					DB_PORT=3306
					DB_DATABASE=laravel
					DB_USERNAME=root
					DB_PASSWORD=
			And then migrate db please.
				<command>: php artisan migrate

		- run
			<command>: php artisan serve

		You can check this project running on http://localhost:8000/.
